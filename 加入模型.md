请根据以下要求生成一个新的二极管体二极管模型文件，用于本工程多模型竞技对比。

1) 工程路径：
   d:\VScode\功率模块_体二极管_20260118
2) 模型目录：
   DiodeFitter/models/
3) 数据约定：

- CSV 列：Vds, Id, Vgs, Tj
- 第三象限：Vds < 0, Id < 0
- 内部定义：Vf = -Vds, If = -Id
- 目标：显式 IofV 形式 Id(Vds, Vgs)

4) 你必须输出一个完整的 MATLAB 文件：
   DiodeFitter/models/model_{{模型名}}.m
5) 模型文件必须返回以下字段（字段名不可改）：
   spec.name
   spec.description
   spec.mode = 'IofV'
   spec.param_names
   spec.p0
   spec.lb
   spec.ub
   spec.equation = @(p, vds, vgs, cfg) ...
   spec.model_fun = spec.equation
   spec.x0_generator = @() ...
   （可选）spec.skip_fit = true
6) 函数要求：

- 支持向量化输入 vds、vgs
- 输出 If >= 0
- 避免指数溢出（可用 min(expo, cfg.model.expo_max)）
- 若用 Vf，请用 vf = -vds
- 不依赖额外工具箱

7) 输出格式示例（只输出完整文件内容，不要其他说明）：
   function spec = model_example(cfg)
   if nargin < 1 || isempty(cfg)
   cfg = config_fit();
   end
   spec.name = 'EXAMPLE';
   spec.description = '示例模型';
   spec.mode = 'IofV';
   spec.param_names = {'a','b','c'};
   spec.p0 = [1, 2, 3];
   spec.lb = [0, 0, 0];
   spec.ub = [10, 10, 10];
   spec.equation = @example_fun;
   spec.model_fun = spec.equation;
   spec.x0_generator = @() spec.p0;
   end

function If = example_fun(p, vds, vgs, cfg)
vf = -vds;
If = p(1) * exp(p(2) * vf) + p(3) * max(vgs, 0);
If = max(If, 0);
end
